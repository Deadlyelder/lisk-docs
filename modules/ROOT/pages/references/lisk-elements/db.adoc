The @liskhq/lisk-db package is a database access implementation for use with Lisk-related software.

== Installation

[source,bash]
----
npm install @liskhq/lisk-db
----

== Update

To perform an upgrade, execute the following command:

[source,bash]
----
npm update @liskhq/lisk-db
----
