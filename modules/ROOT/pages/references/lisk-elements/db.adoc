The @liskhq/lisk-db package providing a database access layer that is a thin wrapper around {url_rocksdb}[RocksDB^].

== Installation

[source,bash]
----
npm install @liskhq/lisk-db
----

== Update

To perform an upgrade, execute the following command:

[source,bash]
----
npm update @liskhq/lisk-db
----
