== Start

[source,bash]
----
pm2 start ecosystem.config.js

# or

npm start
----

== Status

[source,bash]
----
pm2 list
----

== Stop

[source,bash]
----
pm2 stop ecosystem.config.js
----

== Restart

[source,bash]
----
pm2 restart ecosystem.config.js
----

== Remove all processes from pm2 list

[source,bash]
----
pm2 delete ecosystem.config.js

# or

npm stop
----

== Clean all run-time files with dependencies

[source,bash]
----
make clean
----
